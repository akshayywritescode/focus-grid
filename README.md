# Focus Grid

A minimalist web app that helps artists practice using a focused grid system.

Upload an image, divide it into a grid, and isolate one tile at a time to reduce overwhelm and improve focus.

## Features (MVP)

- Upload reference image
- Adjustable rows and columns
- Click to select a tile
- Dim surrounding tiles
- Keyboard navigation (arrow keys)
- Optional tile completion marking

## Tech Stack

- HTML
- CSS
- JavaScript (Canvas API)
- Hosted on GitHub Pages

## Philosophy

Focus Grid is built to reduce friction when starting a drawing.

No accounts.  
No backend.  
No distractions.  

Just focus.
